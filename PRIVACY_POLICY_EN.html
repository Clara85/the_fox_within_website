Privacy Policy

App Information

  • App Name: The fox within (Conscious Focus)
  • Developer: Cheng Zhang
  • Contact Email: thefoxwithinapp@gmail.com
  • App Package Name: com.thefoxwithin.app

Data Collection

This app collects the following data to provide core functionality:

1. User Account Information

  • Email Address: Used for user registration, login, and account management
  • Username: Used for personalized display
  • User ID: System-generated unique identifier

2. App Activity Data

  • Task Data: Daily tasks created and managed by users
  • Notes and Records: Daily reflections, tomorrow's plans, random thoughts, etc.
  • Learning Progress: Course learning records and notes
  • Task Completion Records: Task completion status and satisfaction ratings

3. App Usage Data (Optional)

  • Crash Logs: Used to improve app stability and performance (if crash reporting tools are integrated)

Data Storage

Local Storage

  • All user data is stored locally on the device
  • Uses device security mechanisms to ensure data security

Cloud Storage

  • User data is synchronized to Supabase cloud servers
  • Cloud storage is used for:
    • Multi-device data synchronization
    • Data backup and recovery
    • Ensuring data is not lost

Data Encryption

  • Transmission Encryption: All data transmission uses HTTPS encryption protocol
  • Storage Encryption: Sensitive data is encrypted when stored
  • Access Control: Uses user authentication and authorization mechanisms to ensure only the user can access their own data

Data Usage

Purpose of Use

  • ✅ Core App Functionality: Providing task management, recording, learning features, etc.
  • ✅ Data Synchronization: Synchronizing user data across multiple devices
  • ✅ User Experience Optimization: Improving app functionality and interface
  • ❌ Advertising: This app does not contain ads and will not use data for advertising
  • ❌ Data Analytics: Will not use data for third-party data analytics (unless user explicitly consents)
  • ❌ Personalized Recommendations: Will not use data for personalized recommendation systems

Data Sharing

  • Sharing with Supabase: User data is stored on Supabase cloud servers
    • Supabase follows strict data security standards
    • Supabase will not use user data for other purposes
    • For more information, visit: https://supabase.com/privacy
  • No Third-Party Sharing: Except for Supabase, we do not share user data with any third parties

User Rights

You have the following rights regarding your data:

1. Right to Access

  • You can view all your data in the app at any time

2. Right to Deletion

  • You can delete any data in the app
  • You can clear all data in settings
  • When you delete your account, all related data will be permanently deleted

3. Right to Export

  • You can export all data in settings (JSON format)

4. Right to Modification

  • You can modify your account information and data at any time

5. Right to Withdraw Consent

  • You can withdraw your consent to data collection at any time
  • After withdrawing consent, some features may become unavailable

Data Security Measures

We take the following measures to protect your data security:

  1. Encrypted Transmission: All data transmission uses HTTPS encryption protocol
  2. Encrypted Storage: Sensitive data is encrypted when stored
  3. Access Control: Uses user authentication and authorization mechanisms
  4. Data Isolation: Each user's data is isolated by user ID
  5. Regular Backups: Cloud data is regularly backed up to prevent data loss
  6. Security Audits: Regular security audits and vulnerability scans

Third-Party Services

This app uses the following third-party services:

Supabase

  • Purpose: User authentication and data storage
  • Data Sharing: User account information and app data
  • Privacy Policy: https://supabase.com/privacy
  • Security Measures: Supabase follows industry-standard security measures

Local Notification Services

  • Purpose: Sending task reminders and notifications
  • Data Sharing: Does not share any data, runs only on local device

Data Retention

  • Active Accounts: Data will be retained as long as your account is active
  • Account Deletion: After account deletion, all data will be permanently deleted within 30 days
  • Legal Requirements: In cases required by law, certain data may need to be retained

Children's Privacy

This app is not specifically directed at children under 13 years of age. We do not knowingly collect personal information from children under 13. If you discover that we have inadvertently collected information from a child, please contact us immediately, and we will delete it as soon as possible.

International Data Transfers

Your data may be transferred to and stored on servers located outside your country/region. We ensure that all data transfers comply with applicable data protection laws.

Policy Updates

This privacy policy may be updated periodically. When updated, we will:

  • Notify you within the app
  • Update the "Last Updated" date on this page
  • For significant changes, we will notify you through in-app notifications or email
We recommend that you review this privacy policy regularly to stay informed.

If you are located in the European Union, the legal basis for our processing of your personal data includes:

  • Legitimate Interests: To improve app functionality and user experience
  • Consent: In certain cases, we will seek your explicit consent

United States Privacy Rights (CCPA)

If you are a California resident, you have additional rights under the California Consumer Privacy Act (CCPA):

Your California Privacy Rights

  • Right to Know: You have the right to request information about the personal information we collect, use, disclose, and sell
  • Right to Delete: You have the right to request deletion of your personal information
  • Right to Opt-Out: We do not sell your personal information, so there is no need to opt-out
  • Right to Non-Discrimination: We will not discriminate against you for exercising your privacy rights

How to Exercise Your Rights

To exercise your California privacy rights, please contact us at: thefoxwithinapp@gmail.com

We will respond to your request within 45 days.

Canadian Privacy Rights (PIPEDA)

If you are located in Canada, you have rights under the Personal Information Protection and Electronic Documents Act (PIPEDA):

Your Canadian Privacy Rights

  • Right to Access: You have the right to access your personal information
  • Right to Correction: You have the right to request correction of inaccurate information
  • Right to Withdraw Consent: You can withdraw your consent to data collection at any time
  • Right to File a Complaint: You can file a complaint with the Privacy Commissioner of Canada if you believe your privacy rights have been violated

Data Storage in Canada

We use Supabase for cloud storage, which may store data on servers located outside Canada. We ensure that all data transfers comply with PIPEDA requirements and applicable data protection laws.
