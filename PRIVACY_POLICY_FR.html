Politique de Confidentialité

Informations sur l'Application

  • Nom de l'application : The fox within (Conscious Focus)
  • Développeur : Cheng Zhang
  • Courriel de contact : thefoxwithinapp@gmail.com
  • Nom du package de l'application : com.thefoxwithin.app

Collecte de Données

Cette application collecte les données suivantes pour fournir les fonctionnalités principales :

1. Informations de Compte Utilisateur

  • Adresse courriel : Utilisée pour l'inscription, la connexion et la gestion du compte
  • Nom d'utilisateur : Utilisé pour l'affichage personnalisé
  • ID utilisateur : Identifiant unique généré par le système

2. Données d'Activité de l'Application

  • Données de tâches : Tâches quotidiennes créées et gérées par les utilisateurs
  • Notes et enregistrements : Réflexions quotidiennes, plans pour demain, pensées aléatoires, etc.
  • Progrès d'apprentissage : Enregistrements d'apprentissage de cours et notes
  • Enregistrements d'achèvement de tâches : Statut d'achèvement des tâches et évaluations de satisfaction

3. Données d'Utilisation de l'Application (Optionnel)

  • Journaux de plantage : Utilisés pour améliorer la stabilité et les performances de l'application (si des outils de rapport de plantage sont intégrés)

Stockage des Données

Stockage Local

  • Toutes les données utilisateur sont stockées localement sur l'appareil
  • Utilise les mécanismes de sécurité de l'appareil pour assurer la sécurité des données

Stockage Cloud

  • Les données utilisateur sont synchronisées vers les serveurs cloud Supabase
  • Le stockage cloud est utilisé pour :
    • La synchronisation des données multi-appareils
    • La sauvegarde et la récupération des données
    • Assurer que les données ne sont pas perdues

Chiffrement des Données

  • Chiffrement de transmission : Toutes les transmissions de données utilisent le protocole de chiffrement HTTPS
  • Chiffrement de stockage : Les données sensibles sont chiffrées lors du stockage
  • Contrôle d'accès : Utilise des mécanismes d'authentification et d'autorisation des utilisateurs pour garantir que seul l'utilisateur peut accéder à ses propres données

Utilisation des Données

Objectif d'Utilisation

  • ✅ Fonctionnalités principales de l'application : Fournir des fonctionnalités de gestion de tâches, d'enregistrement, d'apprentissage, etc.
  • ✅ Synchronisation des données : Synchroniser les données utilisateur sur plusieurs appareils
  • ✅ Optimisation de l'expérience utilisateur : Améliorer les fonctionnalités et l'interface de l'application
  • ❌ Publicité : Cette application ne contient pas de publicités et n'utilisera pas les données à des fins publicitaires
  • ❌ Analyse de données : N'utilisera pas les données pour l'analyse de données tierces (sauf consentement explicite de l'utilisateur)
  • ❌ Recommandations personnalisées : N'utilisera pas les données pour les systèmes de recommandation personnalisés

Partage des Données

  • Partage avec Supabase : Les données utilisateur sont stockées sur les serveurs cloud Supabase
    • Supabase suit des normes strictes de sécurité des données
    • Supabase n'utilisera pas les données utilisateur à d'autres fins
    • Pour plus d'informations, visitez : https://supabase.com/privacy
  • Pas de partage avec des tiers : À l'exception de Supabase, nous ne partageons pas les données utilisateur avec des tiers

Droits des Utilisateurs

Vous avez les droits suivants concernant vos données :

1. Droit d'Accès

  • Vous pouvez consulter toutes vos données dans l'application à tout moment

2. Droit de Suppression

  • Vous pouvez supprimer toutes les données dans l'application
  • Vous pouvez effacer toutes les données dans les paramètres
  • Lorsque vous supprimez votre compte, toutes les données associées seront définitivement supprimées

3. Droit d'Exportation

  • Vous pouvez exporter toutes les données dans les paramètres (format JSON)

4. Droit de Modification

  • Vous pouvez modifier vos informations de compte et vos données à tout moment

5. Droit de Retirer le Consentement

  • Vous pouvez retirer votre consentement à la collecte de données à tout moment
  • Après avoir retiré le consentement, certaines fonctionnalités peuvent devenir indisponibles

Mesures de Sécurité des Données

Nous prenons les mesures suivantes pour protéger la sécurité de vos données :

  1. Transmission chiffrée : Toutes les transmissions de données utilisent le protocole de chiffrement HTTPS
  2. Stockage chiffré : Les données sensibles sont chiffrées lors du stockage
  3. Contrôle d'accès : Utilise des mécanismes d'authentification et d'autorisation des utilisateurs
  4. Isolation des données : Les données de chaque utilisateur sont isolées par ID utilisateur
  5. Sauvegardes régulières : Les données cloud sont régulièrement sauvegardées pour éviter la perte de données
  6. Audits de sécurité : Audits de sécurité réguliers et scans de vulnérabilités

Services Tiers

Cette application utilise les services tiers suivants :

Supabase

  • Objectif : Authentification des utilisateurs et stockage des données
  • Partage de données : Informations de compte utilisateur et données de l'application
  • Politique de confidentialité : https://supabase.com/privacy
  • Mesures de sécurité : Supabase suit des mesures de sécurité conformes aux normes de l'industrie

Services de Notification Locale

  • Objectif : Envoi de rappels de tâches et de notifications
  • Partage de données : Ne partage aucune donnée, fonctionne uniquement sur l'appareil local

Conservation des Données

  • Comptes actifs : Les données seront conservées tant que votre compte est actif
  • Suppression de compte : Après la suppression du compte, toutes les données seront définitivement supprimées dans les 30 jours
  • Exigences légales : Dans les cas requis par la loi, certaines données peuvent devoir être conservées

Confidentialité des Enfants

Cette application n'est pas spécifiquement destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment d'informations personnelles auprès d'enfants de moins de 13 ans. Si vous découvrez que nous avons involontairement collecté des informations auprès d'un enfant, veuillez nous contacter immédiatement, et nous les supprimerons dès que possible.

Transferts Internationaux de Données

Vos données peuvent être transférées vers et stockées sur des serveurs situés en dehors de votre pays/région. Nous nous assurons que tous les transferts de données sont conformes aux lois applicables sur la protection des données.

Mises à Jour de la Politique

Cette politique de confidentialité peut être mise à jour périodiquement. Lors des mises à jour, nous :

  • Vous informerons dans l'application
  • Mettre à jour la date de "Dernière mise à jour" sur cette page
  • Pour les changements importants, nous vous informerons par des notifications dans l'application ou par courriel
Nous vous recommandons de consulter régulièrement cette politique de confidentialité pour rester informé.

Nous Contacter

Si vous avez des questions, commentaires ou plaintes concernant cette politique de confidentialité, veuillez nous contacter par :

  • Courriel : thefoxwithinapp@gmail.com
  • Commentaires dans l'application : Soumettre des commentaires dans la page des paramètres de l'application

Base Légale (RGPD)

Si vous êtes situé dans l'Union européenne, la base légale de notre traitement de vos données personnelles comprend :

  • Exécution du contrat : Pour fournir les services de l'application
  • Intérêts légitimes : Pour améliorer les fonctionnalités de l'application et l'expérience utilisateur
  • Consentement : Dans certains cas, nous solliciterons votre consentement explicite

Droits de Confidentialité Canadiens (LPRPDE)

Si vous êtes situé au Canada, vous avez des droits en vertu de la Loi sur la protection des renseignements personnels et les documents électroniques (LPRPDE) :

Vos Droits de Confidentialité Canadiens

  • Droit d'accès : Vous avez le droit d'accéder à vos renseignements personnels
  • Droit de correction : Vous avez le droit de demander la correction d'informations inexactes
  • Droit de retirer le consentement : Vous pouvez retirer votre consentement à la collecte de données à tout moment
  • Droit de déposer une plainte : Vous pouvez déposer une plainte auprès du Commissaire à la protection de la vie privée du Canada si vous croyez que vos droits de confidentialité ont été violés

Stockage des Données au Canada

Nous utilisons Supabase pour le stockage cloud, qui peut stocker des données sur des serveurs situés à l'extérieur du Canada. Nous nous assurons que tous les transferts de données sont conformes aux exigences de la LPRPDE et aux lois applicables sur la protection des données.

Conformité avec la Loi sur la Protection des Consommateurs du Québec

Si vous êtes résident du Québec, vous avez également des droits supplémentaires en vertu de la Loi sur la protection des consommateurs du Québec. Nous nous engageons à respecter toutes les lois applicables du Québec concernant la protection des renseignements personnels.
