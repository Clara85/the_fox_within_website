# Privacy Policy

- **App Name**: The fox within (Conscious Focus)

- **Contact Email**: thefoxwithinapp@gmail.com

## Data Collection

This app collects the following data to provide core functionality:

### 1. User Account Information
- **Email Address**: Used for user registration, login, and account management
- **Username**: Used for personalized display
- **User ID**: System-generated unique identifier

### 2. App Activity Data
- **Task Data**: Daily tasks created and managed by users
- **Notes and Records**: Daily reflections, tomorrow's plans, random thoughts, etc.
- **Learning Progress**: Course learning records and notes
- **Task Completion Records**: Task completion status and satisfaction ratings

### 3. App Usage Data (Optional)
- **Crash Logs**: Used to improve app stability and performance (if crash reporting tools are integrated)

## Data Storage

### Local Storage
- All user data is stored locally on the device
- Uses device security mechanisms to ensure data security

### Cloud Storage
- User data is synchronized to Supabase cloud servers
- Cloud storage is used for:
  - Multi-device data synchronization
  - Data backup and recovery
  - Ensuring data is not lost

### Data Encryption
- **Transmission Encryption**: All data transmission uses HTTPS encryption protocol
- **Storage Encryption**: Sensitive data is encrypted when stored
- **Access Control**: Uses user authentication and authorization mechanisms to ensure only the user can access their own data

## Data Usage

### Purpose of Use
- ✅ **Core App Functionality**: Providing task management, recording, learning features, etc.
- ✅ **Data Synchronization**: Synchronizing user data across multiple devices
- ✅ **User Experience Optimization**: Improving app functionality and interface
- ❌ **Advertising**: This app does not contain ads and will not use data for advertising
- ❌ **Data Analytics**: Will not use data for third-party data analytics (unless user explicitly consents)
- ❌ **Personalized Recommendations**: Will not use data for personalized recommendation systems

### Data Sharing
- **Sharing with Supabase**: User data is stored on Supabase cloud servers
  - Supabase follows strict data security standards
  - Supabase will not use user data for other purposes
  - For more information, visit: https://supabase.com/privacy

- **No Third-Party Sharing**: Except for Supabase, we do not share user data with any third parties

## User Rights

You have the following rights regarding your data:

### 1. Right to Access
- You can view all your data in the app at any time

### 2. Right to Deletion
- You can delete any data in the app
- You can clear all data in settings
- When you delete your account, all related data will be permanently deleted

### 3. Right to Export
- You can export all data in settings (JSON format)

### 4. Right to Modification
- You can modify your account information and data at any time

### 5. Right to Withdraw Consent
- You can withdraw your consent to data collection at any time
- After withdrawing consent, some features may become unavailable

## Data Security Measures

We take the following measures to protect your data security:

1. **Encrypted Transmission**: All data transmission uses HTTPS encryption protocol
2. **Encrypted Storage**: Sensitive data is encrypted when stored
3. **Access Control**: Uses user authentication and authorization mechanisms
4. **Data Isolation**: Each user's data is isolated by user ID
5. **Regular Backups**: Cloud data is regularly backed up to prevent data loss
6. **Security Audits**: Regular security audits and vulnerability scans
